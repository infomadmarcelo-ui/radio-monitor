# STR Monitor

Sistema de monitoreo de streaming de radio en tiempo real.

## Instalación como PWA

Una vez publicado en GitHub Pages, abrí la URL en el dispositivo y buscá la opción "Agregar a pantalla de inicio" o "Instalar app".

## Archivos

- `index.html` — aplicación principal
- `manifest.json` — configuración PWA
- `sw.js` — service worker (modo offline)
- `icons/` — íconos para todos los dispositivos
